# X投稿テンプレート(本文+自己リプで伸ばす)

記事(note/Zenn)を出したときのX投稿の型。インプ(表示)を伸ばすための固定フォーマット。毎回ここを開いてスロット(`{ }`)を埋めて使う。方針の背景は `README.md` の「運用ルール」を参照。

## 基本原則(なぜこの型か)

- **本文は単体で完結した価値にする。外部リンク(note/Zenn)は本文に貼らず、自己リプ(返信)に回す。** Xは外部リンクのある投稿のリーチを下げるため。
- **ハッシュタグは1〜2個まで。** Xではタグはほぼリーチに効かず、多用は宣伝臭くて逆効果。
- **画像を1枚付ける**(週次サムネ・グラフ・コードのbefore/afterスクショなど)。滞在時間が伸びて拡散されやすい。
- **付いた反応(リプ・引用)には全部返信する。** 会話がいちばん評価される。リンクを撒くより会話で伸ばす。

## 投稿の型(3段)

1. **本投稿**: 【フック1行】+【中身2〜4行】+【画像】+【#タグ1〜2】
2. **自己リプ①**: 記事リンク(「詳細/続きはこちら👇」+ URL)
3. **自己リプ②(任意)**: シリーズの背景 or 前回記事へのリンク(初見の人を過去記事へ誘導)

### フック1行の作り方

先頭に**具体・数字・痛み・結果**のどれかを置く。1行目でスクロールを止められないと以降は読まれない。

- 悪い例: 「ブログ書きました」「週次レポート更新しました」
- 良い例: 「新規ユーザーだけ永久ローディングになる罠にハマった」「はじめの1週間、体重は−0.23kg動いた」

## 記事タイプ別テンプレ

### A. note(週次レポート・一般向け)

本投稿:

```
自作アプリで減量実験、今週の結果です📊

{今週の一番の数字。例: 週平均 −0.23kg / 記録率100% / 連続78日}
{一言の所感 or AIコーチ判定。例: AIコーチには「やや遅れ」と言われつつ}

10/31まで−8kg、淡々と。
#個人開発
```

画像: その週の週次サムネ(`note0N_サムネ.png`)

自己リプ①:

```
詳しい数字とAIコーチのコメントはこちら👇
{note記事URL}
```

自己リプ②(任意・初見向けの背景導線):

```
そもそもなぜ健康管理アプリを自作したのか、という背景はこちら👇
https://note.com/yotti073/n/n54d80318b1d8
```

### B. Zenn(技術記事・エンジニア向け)

本投稿:

```
{技術的な痛み/gotchaを1行。例: 開発中は自分のブラウザにデータがあるから絶対踏まないけど、新規ユーザーだけ永久ローディングになる罠がある}

{2〜3行で中身の要点。数字や固有名詞(Dexie/useLiveQueryなど)を入れる}

原因と対処をZennに書きました。
#個人開発
```

画像: コードのbefore/after(❌/✅)スクショ1枚

自己リプ①:

```
記事はこちら👇
{Zenn記事URL}
```

自己リプ②(任意・シリーズ導線):

```
開発の進め方(仕様書駆動 × Claude Code)はこちら👇
https://zenn.dev/yotti073/articles/spec-driven-claude-code
```

## 投稿前後のチェックリスト

- [ ] 投稿はフォロワーが起きている時間(平日 朝7〜9時 / 昼 / 夜21〜23時)。技術記事は平日朝が伸びやすい。
- [ ] 投稿の15〜30分前に、同じ界隈の他人のポストへリプして自分を温める(初速が付きやすい)。
- [ ] 本文にURLを入れていない(リンクは自己リプに回した)。
- [ ] ハッシュタグは1〜2個。
- [ ] 画像を1枚付けた。
- [ ] フック1行が「具体・数字・痛み・結果」のどれかになっている。
- [ ] 読者層で混ぜていない(一般向けnoteの投稿にエンジニア向けZenn記事をぶら下げない、など)。
- [ ] 投稿後、付いた反応には全部返信して会話を続ける。

## 発信のレーン(軸)

「**自作アプリで自分を人体実験している個人開発者**」(開発 × 減量 × AI × 実データ)という掛け算で一貫させる。bio・固定ポストもこの軸に揃えると、プロフィールに来た人がフォローしやすい。

## 補足: 過去投稿への後追いリプについて

すでに投稿して時間が経った(=初速の終わった)ポストに、あとから自己リプでリンクを足しても、届くのは「スレッドを開いた人」だけで表示数は伸びにくい。**永久的な導線としては足す価値があるが、伸びは期待しすぎない。** 次回からは本投稿と同時に自己リプまで一気に組んで、最初からスレッドにするのが伸びる形。
